# 1. 목적 및 필요성 (Maqsad va Zarurat)

## 1.1 작품 개발 목적 (Loyiha Ishlab Chiqish Maqsadi)

### 한국어:
```
본 작품인 Personal Leak Detector (PLD)는 사용자의 개인정보가 이메일이나 파일을 통해 어디에 
노출되었는지 자동으로 탐지하고, 피싱 이메일을 식별하여 사용자에게 경고하는 종합적인 보안 솔루션을 
제공하는 것을 목적으로 합니다.

**주요 개발 목적:**

1. **개인정보 유출 자동 탐지**
   - 이메일 내 개인정보(PII) 자동 탐지
   - 파일 내 개인정보 탐지
   - 실시간 모니터링 및 경고

2. **피싱 공격 방지**
   - 피싱 이메일 조기 탐지
   - 위험 점수 기반 분류
   - 사용자 경고 시스템

3. **사용자 편의성 제공**
   - 자동화된 스캔 프로세스
   - 직관적인 대시보드
   - 다국어 지원

4. **보안 인식 제고**
   - 개인정보 보호 중요성 인식
   - 사이버 보안 교육 도구
   - 디지털 프라이버시 보호
```

### O'zbekcha:
```
Ushbu loyiha Personal Leak Detector (PLD) foydalanuvchilarning shaxsiy ma'lumotlarining email yoki 
fayllar orqali qayerda oqib ketganini avtomatik aniqlash va phishing email'larni identifikatsiya 
qilib, foydalanuvchilarni ogohlantiruvchi umumiy xavfsizlik yechimini taqdim etishni maqsad qiladi.

**Asosiy ishlab chiqish maqsadlari:**

1. **Shaxsiy ma'lumotlar oqib ketishini avtomatik aniqlash**
   - Email ichidagi shaxsiy ma'lumotlarni (PII) avtomatik aniqlash
   - Fayl ichidagi shaxsiy ma'lumotlarni aniqlash
   - Real vaqtda monitoring va ogohlantirish

2. **Phishing hujumlarini oldini olish**
   - Phishing email'larni erta aniqlash
   - Xavf balli asosidagi tasniflash
   - Foydalanuvchi ogohlantirish tizimi

3. **Foydalanuvchi qulayligini ta'minlash**
   - Avtomatlashtirilgan skanerlash jarayoni
   - Intuitiv dashboard
   - Ko'p tillilik qo'llab-quvvatlash

4. **Xavfsizlik tushunchasini oshirish**
   - Shaxsiy ma'lumotlarni himoya qilish muhimligini tushunish
   - Kiberxavfsizlik ta'lim vositasi
   - Raqamli maxfiylikni himoya qilish
```

## 1.2 개발 필요성 (Ishlab Chiqish Zarurati)

### 1.2.1 문제 상황 분석 (Muammo Vaziyatini Tahlil Qilish)

### 한국어:
```
**현재 개인정보 유출 문제의 심각성:**

1. **개인정보 유출 사고 증가**
   - 2023년 전 세계적으로 4,100건 이상의 데이터 유출 사고 발생
   - 평균 데이터 유출 비용: $4.45 million (2023년 기준)
   - 개인정보 유출로 인한 피해자 수: 수억 명

2. **피싱 공격의 증가**
   - 2023년 피싱 공격 증가율: 47%
   - 평균 피싱 이메일 클릭률: 3.2%
   - 피싱으로 인한 연간 손실: $10 billion 이상

3. **기존 솔루션의 한계**
   - 실시간 이메일 스캔 기능 부재
   - 개인 사용자 대상 솔루션 부족
   - 비용이 높은 기업용 솔루션 위주
   - 사용자 친화적 인터페이스 부족

4. **사용자 인식 부족**
   - 개인정보 유출 위험 인식 부족
   - 피싱 이메일 식별 능력 부족
   - 보안 대응 지식 부족
```

### O'zbekcha:
```
**Hozirgi shaxsiy ma'lumotlar oqib ketish muammosining jiddiyligi:**

1. **Shaxsiy ma'lumotlar oqib ketish hodisalari ko'payishi**
   - 2023 yilda butun dunyo bo'ylab 4,100+ data breach hodisalari yuz berdi
   - O'rtacha data breach xarajati: $4.45 million (2023 yil mezonida)
   - Shaxsiy ma'lumotlar oqib ketishi natijasidagi jabrlanuvchilar soni: yuz millionlab kishilar

2. **Phishing hujumlarining ko'payishi**
   - 2023 yilda phishing hujumlarining o'sish darajasi: 47%
   - O'rtacha phishing email bosish foizi: 3.2%
   - Phishing natijasidagi yillik zarar: $10 billion dan ko'p

3. **Mavjud yechimlarning cheklovlari**
   - Real vaqtda email skanerlash funksiyasi yo'q
   - Shaxsiy foydalanuvchilar uchun yechimlar yetishmaydi
   - Qimmat korxona yechimlari asosiy o'rinni egallaydi
   - Foydalanuvchiga qulay interfeys yetishmaydi

4. **Foydalanuvchi tushunchasining yetishmasligi**
   - Shaxsiy ma'lumotlar oqib ketish xavfi tushunchasining yetishmasligi
   - Phishing email'larni aniqlash qobiliyatining yetishmasligi
   - Xavfsizlik javob berish bilimlarining yetishmasligi
```

### 1.2.2 기존 솔루션의 한계 (Mavjud Yechimlarning Cheklovlari)

### 한국어:
```
**기존 솔루션 분석:**

1. **Have I Been Pwned (HIBP)**
   - 장점: 무료, 알려진 데이터 유출 검색
   - 한계: 실시간 스캔 불가, PII 탐지 없음, 피싱 탐지 없음

2. **Google Advanced Protection Program**
   - 장점: 강력한 보안 기능
   - 한계: Google 계정만 지원, 사용자 스캔 기능 없음, 비용 발생

3. **기업용 보안 솔루션**
   - 장점: 포괄적인 보안 기능
   - 한계: 높은 비용, 개인 사용자 부적합, 복잡한 설정

4. **오픈소스 도구**
   - 장점: 무료, 커스터마이징 가능
   - 한계: 기술적 지식 필요, 통합 솔루션 부족
```

### O'zbekcha:
```
**Mavjud yechimlar tahlili:**

1. **Have I Been Pwned (HIBP)**
   - Afzalliklari: Bepul, ma'lum data breach qidiruv
   - Cheklovlari: Real vaqtda skanerlash mumkin emas, PII aniqlash yo'q, phishing aniqlash yo'q

2. **Google Advanced Protection Program**
   - Afzalliklari: Kuchli xavfsizlik funksiyalari
   - Cheklovlari: Faqat Google hisoblarini qo'llab-quvvatlaydi, foydalanuvchi skanerlash funksiyasi yo'q, xarajatlar

3. **Korxona xavfsizlik yechimlari**
   - Afzalliklari: Keng qamrovli xavfsizlik funksiyalari
   - Cheklovlari: Yuqori xarajatlar, shaxsiy foydalanuvchilar uchun mos emas, murakkab sozlash

4. **Open-source vositalar**
   - Afzalliklari: Bepul, sozlash mumkin
   - Cheklovlari: Texnik bilimlar kerak, integratsiyalangan yechim yetishmaydi
```

## 1.3 사회적 수요 (Ijtimoiy Talab)

### 1.3.1 개인 사용자 수요 (Shaxsiy Foydalanuvchilar Talabi)

### 한국어:
```
**개인 사용자의 보안 수요 증가:**

1. **디지털화 가속화**
   - 온라인 활동 증가로 개인정보 노출 위험 증가
   - 이메일 사용량 증가 (일일 평균 126개 이메일)
   - 클라우드 저장소 사용 증가

2. **개인정보 보호 인식 향상**
   - GDPR, 개인정보보호법 등 규제 강화
   - 개인정보 유출 사고에 대한 관심 증가
   - 프라이버시 보호 의식 제고

3. **피싱 공격 피해 증가**
   - 개인 사용자 대상 피싱 공격 증가
   - 금전적 피해 증가
   - 신원 도용 위험 증가

**수요 통계:**
- 개인정보 보호 솔루션 시장 성장률: 연평균 15%
- 개인 사용자 보안 소프트웨어 사용률: 65% (2023년)
- 피싱 방지 도구 수요: 연평균 25% 증가
```

### O'zbekcha:
```
**Shaxsiy foydalanuvchilarning xavfsizlik talabining o'sishi:**

1. **Raqamlashtirish tezligining oshishi**
   - Onlayn faoliyatning o'sishi natijasida shaxsiy ma'lumotlar oqib ketish xavfi oshmoqda
   - Email ishlatishning o'sishi (kunlik o'rtacha 126 ta email)
   - Cloud saqlash ishlatishining o'sishi

2. **Shaxsiy ma'lumotlarni himoya qilish tushunchasining oshishi**
   - GDPR, shaxsiy ma'lumotlarni himoya qilish qonuni va boshqa qoidalarning kuchayishi
   - Shaxsiy ma'lumotlar oqib ketish hodisalariga qiziqishning o'sishi
   - Maxfiylikni himoya qilish tushunchasining oshishi

3. **Phishing hujumlaridan zarar ko'rishning o'sishi**
   - Shaxsiy foydalanuvchilarga qaratilgan phishing hujumlarining o'sishi
   - Moliyaviy zararning o'sishi
   - Shaxsni o'g'irlash xavfining o'sishi

**Talab statistikalari:**
- Shaxsiy ma'lumotlarni himoya qilish yechimlari bozorining o'sish darajasi: Yillik o'rtacha 15%
- Shaxsiy foydalanuvchi xavfsizlik dasturiy ta'minoti ishlatish darajasi: 65% (2023 yil)
- Phishing oldini olish vositalari talabi: Yillik o'rtacha 25% o'sish
```
