## 슬라이드 3: 프로젝트 개요
# 프로젝트 개요

## Personal Leak Detector (PLD)

**프로젝트 목적**
- 사용자의 이메일, 파일, 웹 콘텐츠에서 개인정보 유출을 자동으로 탐지
- 피싱 이메일을 실시간으로 식별 및 차단
- 위험도 점수를 통한 종합적인 보안 평가

**주요 기능**
1. Gmail 이메일 자동 스캔 (최대 2,000개)
2. 다양한 PII (개인식별정보) 탐지
3. 피싱 이메일 탐지 및 위험도 평가
4. 실시간 대시보드 및 시각화
5. 파일 업로드 및 분석

**중요 참고사항**
- **탐지 방법**: 규칙 기반 (rule-based) 알고리즘
- **Regular Expression**: 패턴 매칭
- **휴리스틱 알고리즘**: 피싱 탐지용
- **AI/ML**: 현재 사용되지 않음 (향후 계획됨)

## bcrypt 해싱

### 비밀번호 해싱
```python
import bcrypt

def get_password_hash(password: str) -> str:
    password_bytes = password.encode('utf-8')
    if len(password_bytes) > 72:
        password_bytes = password_bytes[:72]  # bcrypt 제한
    salt = bcrypt.gensalt()
    hashed = bcrypt.hashpw(password_bytes, salt)
    return hashed.decode('utf-8')
```

### 비밀번호 검증
```python
def verify_password(plain_password: str, hashed_password: str) -> bool:
    password_bytes = plain_password.encode('utf-8')
    if len(password_bytes) > 72:
        password_bytes = password_bytes[:72]
    return bcrypt.checkpw(password_bytes, hashed_password.encode('utf-8'))
```

- Salt 자동 생성
- 72바이트 제한 처리

## 슬라이드 34: 보안 구현 - 토큰 암호화

## OAuth 토큰 암호화

### 암호화 구현
```python
from cryptography.fernet import Fernet
import base64
import hashlib

def encrypt_data(data: str) -> str:
    key = settings.ENCRYPTION_KEY.encode()
    # 32바이트 키 생성
    key_hash = hashlib.sha256(key).digest()
    f = Fernet(base64.urlsafe_b64encode(key_hash))
    encrypted = f.encrypt(data.encode())
    return encrypted.decode()

def decrypt_data(encrypted_data: str) -> str:
    key = settings.ENCRYPTION_KEY.encode()
    key_hash = hashlib.sha256(key).digest()
    f = Fernet(base64.urlsafe_b64encode(key_hash))
    decrypted = f.decrypt(encrypted_data.encode())
    return decrypted.decode()
```

**암호화 알고리즘**: AES-256 (Fernet)

## 슬라이드 37: 핵심 기능 구현 - 개인정보 탐지
# 핵심 기능 구현: 개인정보 탐지

## DetectionEngine 클래스

**중요**: 현재 시스템은 **규칙 기반 (rule-based)** 탐지를 사용하며, AI/ML이 아님.

**참고**: 코드에 NER (Named Entity Recognition)에 대한 언급이 있지만 (`self.ner_model = None`), 실제로는 NER 모델이 로드되지 않고 사용되지 않음. 오직 **Regular Expression 패턴 매칭**만 사용됨.

### 패턴 기반 탐지 (Regular Expression)
```python
import re
from app.models.sensitive_item import PIIType

class DetectionEngine:
    def __init__(self):
        self.patterns = self._init_patterns()
        self.ner_model = None  # 현재 사용되지 않음
    
    def _init_patterns(self):
        """다양한 PII 유형에 대한 regex 패턴 생성"""
        patterns = {
            PIIType.EMAIL.value: [
                (r'\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Z|a-z]{2,}\b', 
                 re.IGNORECASE)
            ],
            PIIType.PHONE.value: [
                (r'\+?\d{1,3}?[-.\s]?\(?\d{1,4}\)?[-.\s]?\d{1,4}[-.\s]?\d{1,4}[-.\s]?\d{1,9}', 
                 re.IGNORECASE),
                (r'\(?\d{3}\)?[-.\s]?\d{3}[-.\s]?\d{4}', re.IGNORECASE)
            ],
            PIIType.CREDIT_CARD.value: [
                (r'\b(?:\d[ -]*?){13,16}\b', re.IGNORECASE)
            ]
        }
        # 패턴 컴파일
        compiled = {}
        for pii_type, pattern_list in patterns.items():
            compiled[pii_type] = [
                (pattern, re.compile(pattern, flags=flags))
                for pattern, flags in pattern_list
            ]
        return compiled
    
    def detect_in_text(self, text: str, sensitive_items=None):
        results = []
        for pii_type, pattern_list in self.patterns.items():
            for pattern_str, compiled_pattern in pattern_list:
                for match in compiled_pattern.finditer(text):
                    value = match.group()
                    
                    # Luhn 알고리즘 검증 (신용카드)
                    if pii_type == PIIType.CREDIT_CARD.value:
                        if not self._luhn_check(value):
                            continue
                    
                    # 컨텍스트 추출
                    snippet = self._extract_context(
                        text, match.start(), match.end()
                    )
                    
                    results.append(DetectionResult(
                        type=pii_type,
                        value=value,
                        snippet=snippet,
                        confidence=0.8
                    ))
        return results
```

## 슬라이드 38: 핵심 기능 구현 - Luhn 알고리즘
# 핵심 기능 구현: Luhn 알고리즘

## 신용카드 번호 검증

### Luhn 알고리즘 구현
```python
def _luhn_check(self, card_number: str) -> bool:
    """Luhn 알고리즘을 사용한 신용카드 번호 검증"""
    digits = re.sub(r'\D', '', card_number)
    
    if len(digits) < 13 or len(digits) > 19:
        return False
    
    total = 0
    reverse_digits = digits[::-1]
    
    for i, digit in enumerate(reverse_digits):
        n = int(digit)
        if i % 2 == 1:  # 짝수 위치 (0-indexed)
            n *= 2
            if n > 9:
                n -= 9
        total += n
    
    return total % 10 == 0
```

**작동 원리**
1. 오른쪽에서 왼쪽으로, 짝수 위치 숫자를 2배
2. 9를 초과하면 9를 빼기
3. 모든 숫자 합계가 10의 배수인지 확인

## 슬라이드 39: 핵심 기능 구현 - 피싱 탐지
# 핵심 기능 구현: 피싱 탐지

## PhishingDetector 클래스

**중요**: 피싱 탐지는 **규칙 기반 (rule-based)** 알고리즘을 사용하며, AI/ML이 아님.

### 위험도 계산 (휴리스틱 알고리즘)
```python
import re
import urllib.parse

class PhishingDetector:
    URGENT_KEYWORDS = [
        r'\burgent\b', r'\bimmediately\b', r'\bverify\s+now\b',
        r'\baccount\s+locked\b', r'\bsuspended\b', r'\bexpired\b'
    ]
    
    SHORTENER_DOMAINS = [
        'bit.ly', 't.co', 'tinyurl.com', 'goo.gl', 'ow.ly'
    ]
    
    def __init__(self):
        self.urgent_patterns = [
            re.compile(pattern, re.IGNORECASE) 
            for pattern in self.URGENT_KEYWORDS
        ]
    
    def detect_phishing(self, message: Dict) -> Dict:
        risk_score = 0.0
        flags = []
        details = {}
        
        # 전체 텍스트 결합
        full_text = ' '.join([
            str(message.get('subject', '') or ''),
            str(message.get('from', '') or ''),
            str(message.get('body', '') or '')
        ]).lower()
        
        # 1. 긴급성 언어 탐지
        urgent_count = sum(1 for pattern in self.urgent_patterns 
                          if pattern.search(full_text))
        if urgent_count > 0:
            flags.append('urgent_language')
            risk_score += min(urgent_count * 5, 20)
            details['urgent_keywords'] = urgent_count
        
        # 2. 단축 URL 탐지
        urls = self._extract_urls(full_text)
        shortened_urls = [url for url in urls 
                         if self._is_shortened_url(url)]
        if shortened_urls:
            flags.append('shortened_urls')
            risk_score += min(len(shortened_urls) * 10, 25)
            details['shortened_urls'] = shortened_urls
        
        # 3. SPF/DKIM/DMARC 검증
        headers = message.get('headers', [])
        spf_result = self._check_spf(headers)
        dkim_result = self._check_dkim(headers)
        dmarc_result = self._check_dmarc(headers)
        
        if spf_result == 'fail' or dkim_result == 'fail':
            flags.append('authentication_failed')
            risk_score += 25
            details['authentication'] = {
                'spf': spf_result,
                'dkim': dkim_result,
                'dmarc': dmarc_result
            }
        
        # 4. 위험도 판정
        is_phishing = risk_score >= 30
        
        return {
            'is_phishing': is_phishing,
            'risk_score': min(risk_score, 100),
            'flags': flags,
            'details': details
        }
```

```python
    def extract_text(file_path: str) -> Optional[str]:
        ext = Path(file_path).suffix.lower()
        file_type = FileProcessor.SUPPORTED_EXTENSIONS.get(ext)
        
        if file_type == 'pdf':
            return FileProcessor._extract_from_pdf(file_path)
        elif file_type == 'docx':
            return FileProcessor._extract_from_docx(file_path)
        elif file_type == 'image':
            return FileProcessor._extract_from_image(file_path)  # OCR
        # ...
```

### PDF 처리 (PyMuPDF)
```python
def _extract_from_pdf(file_path: str) -> str:
    import fitz  # PyMuPDF
    doc = fitz.open(file_path)
    text_parts = []
    for page in doc:
        text_parts.append(page.get_text())
    return '\n'.join(text_parts)
```
